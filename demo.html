如何完成设置

要完成 WebAssembly 项目并在浏览器中运行，请执行以下步骤：

  1. 安装 wasm-pack: cargo install wasm-pack
  2. 构建项目: wasm-pack build --target web
  3. 启动本地服务器: python -m http.server 8000 或使用其他静态文件服务器
  4. 在浏览器中访问 http://localhost:8000

代码说明

Rust 代码特点：

  • #[wasm_bindgen]: 标记需要导出到 JavaScript 的函数
  • console_log!: 自定义宏，用于在浏览器控制台输出调试信息
  • crate-type = ["cdylib"]: 配置生成动态链接库
  • 支持基础数据类型和字符串在 Rust 和 JavaScript 之间传递
